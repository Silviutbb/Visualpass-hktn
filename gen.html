Generatorul VisualPass este o unealta simpla dar eficienta care ofera un plus de securitate. El functioneaza pe principiul compunerii unor parole pe care utilizatorul doar le deseneaza pe tastatura, astfel fiind nevoit sa retina doar forma nu si sirul de caractere in sine. Desi algoritmul este simplist, parolele generate de acesta nu se gasesc in dictionarele de parole ale hackerilor fapt ce sporeste securitatea datelor personale ale utilizatorului de rand.

Pentru implementarea acestui proiect a fost folosit limbajul PHP, HTML5, JavaScript si CSS. Dimensiunile parolelor generate sunt intre 5 si 16 caractere desi valoarea predefinita (in cazul in care utilizatorul nu doreste sa introduca o dimensiune specifica) este de 6 caractere. Algoritmul produce coduri formate atat din litere cat si din cifre.

Acest proiect este open-source, cu liceenta Apache 2 si codul sursa este gazduit pe serverele GitHub. Pentru a avea acces la codul sursa trebuie doar sa accesati linkul : https://github.com/Silviutbb/Visualpass-hktn.git
